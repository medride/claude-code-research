## How to Use

- Each feature has its own `master-plan.md` with detailed tracking
- **Session start**: Read this index, then read the relevant `master-plan.md` for your feature
- **During work**: Update YOUR feature's `master-plan.md`, not this index
- **Update this index only** when adding or removing features/workstreams

## Adding a New Feature

```markdown
| New Feature Name | `tasks/[domain]/master-plan.md` | feature/branch-name | In Progress |
```
